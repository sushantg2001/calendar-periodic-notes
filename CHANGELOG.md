# Changelog

## 1.1.0

- Add a **Create periodic note…** command that opens a quick-add input dialog. Pick a granularity and type `today`, a relative offset (`+1`, `-2`) in that granularity's unit, or a date like `2024-03-15`; a live preview shows the resolved note path, and pressing Enter creates (or opens) the note.

## 1.0.0

- Initial release: calendar view combined with daily, weekly, monthly, quarterly, and yearly periodic notes.
